Provenance

This ontology was made in 2022 ky KurrawongAI for use in large graph management.

Description

This ontology, a data model, defines types of RDF graphs that can be used to segment and then manage other graphs

Machine-readable version:

In RDF (Turtle) format: https://pid.kurrawong.ai/vg or via Content Negotiation

Classes

Real Graph c

IRI https://pid.kurrawong.ai/vg/RealGraph
Is Defined By Virtual Graph Ontology
Description

An RDF Named Graph that may contain triples
Example 
        PREFIX vg: <https://pid.kurrawong.ai/vg/>
        PREFIX sdo: <https://schema.org>

        <http://example/graph/A>
            a vg:RealGraph
            sdo:name "Real Graph A" ;
        .

Virtual Graph c

IRI https://pid.kurrawong.ai/vg/VirtualGraph
Is Defined By Virtual Graph Ontology
Description

A Graph that is an alias for other Graphs and may not contain triples
Example 
        PREFIX vg: <https://pid.kurrawong.ai/vg/>
        PREFIX sdo: <https://schema.org>

        <http://example/graph/X>
            a vg:VirtualGraph
            sdo:name "Virtual Graph X" ;
        .

Object Properties

is alias for op

IRI https://pid.kurrawong.ai/vg/isAliasFor
Is Defined By Virtual Graph Ontology
Description

A Virtual Graph is an alias for a Graph
Example 
        PREFIX vg: <https://pid.kurrawong.ai/vg/>

        # Virtual Graph X is an alias for Real Graphs A & B
        <http://example/graph/X>
            vg:isAliasFor
                 <http://example/graph/A> ,
                 <http://example/graph/B> ;
        .
